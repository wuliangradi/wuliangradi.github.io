工具集

  1. git Git[1] is a version control system (VCS) for tracking changes in computer files and coordinating work on those files among multiple people. Git was created by Linus Torvalds in 2005. As a distributed revision control system it is aimed at speed, data integrity, and support for distributed, non-linear workflows.
  2. github GitHub[2] is a web-based Git or version control repository and Internet hosting service.
  3. markdown Markdown[3] is a lightweight markup language with plain text formatting syntax.
  4. jeykll 将纯文本转换为静态博客网站.GitHub Pages基于Jekyll构建。
  5. mathjax MathJax允许你在你的网页中包含公式，无论是使用LaTeX、MathML或者AsciiMath符号，这些公式都会被javascript处理为HTML、SVG或者MathML符号Markdown[4]。
  6. hexo Hexo 是一个快速、简洁且高效的博客框架[6]。
