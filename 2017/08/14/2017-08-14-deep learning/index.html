第二阶段 双系统安装

  • 进入Ubuntu U盘安装时出现黑屏解决办法
    “e”进入编辑模式，删除”–”， 加上”nomodeset”
  • 安装Ubuntu时屏幕分辨率问题
    • /etc/default/grub
    • 修改#GRUB_GFXMODE=640x480
  • 彻底解决黑屏
    • /etc/default/grub
    • 修改为GRUB_CMDLINE_LINUX_DEFAULT="quiet splash nomodeset"
    • sudo update-grub

第三阶段 CUDA和CUDNN安装

  • 显卡驱动安装

    • sudo apt-get update
